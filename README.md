### palette-picker

## Ryan Harrington

This is my first full stack application. Allows user to randomly generate colors. Lock selected colors, continue replacing colors not wanted. Then store them.

Tech stack:
JavaScript
jQuery
Postgres
Node.js
Knex
Express
